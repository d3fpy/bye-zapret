# bye-zapret

Консольная утилита для Windows, полностью удаляющая **zapret** и связанные с ним компоненты (winws.exe, WinDivert), а также сбрасывающая сетевой стек системы.

## Возможности

- Остановка и удаление службы `zapret`
- Завершение процессов `winws.exe`
- Выгрузка и удаление драйверов `WinDivert` / `WinDivert14`
- Очистка кэша DNS и ARP
- Сброс IP-стека и Winsock
- Проверка прав администратора перед запуском
- Финальный отчёт со статусами всех компонентов
- Легкая .bat версия без установки доп. ПО
## Требования

- Windows 10 / 11
- Запуск **от имени администратора**
- Поддержка ANSI/VT100 в консоли (включена по умолчанию в современных версиях Windows)
