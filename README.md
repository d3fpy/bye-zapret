# bye-zapret

Консольная утилита для Windows, полностью удаляющая **zapret** и связанные с ним компоненты (winws.exe, WinDivert), а также сбрасывающая сетевой стек системы.

## Возможности

- **Полная деинсталляция:** Остановка и удаление службы `zapret`.
- **Очистка процессов:** Принудительное завершение всех запущенных процессов `winws.exe`.
- **Удаление драйверов:** Выгрузка и безопасное удаление драйверов `WinDivert` / `WinDivert14`.
- **Сброс сети:** Очистка кэша DNS/ARP, а также полный сброс IP-стека и Winsock.
- **Безопасность:** Автоматическая проверка прав администратора перед стартом.
- **Информативность:** Финальный детальный отчёт со статусами всех компонентов.
- **Альтернатива:** Легкая `.bat` версия без необходимости установки дополнительного ПО.

## Требования

- Windows 10 / 11
- Запуск **от имени администратора**
- Поддержка ANSI/VT100 в консоли (включена по умолчанию в современных версиях Windows)

## Сборка

Для компиляции C++ версии утилиты выполните команду:

```bash
g++ -std=c++17 -O2 to.cpp -o bye-zapret.exe -lpthread
```

## Использование

1. Скачайте последний релиз утилиты (`bye-zapret.exe` или `.bat` скрипт).
2. Нажмите на файл правой кнопкой мыши и выберите **«Запуск от имени администратора»**.
3. Дождитесь завершения всех этапов очистки и ознакомьтесь с финальным отчётом.
4. Перезагрузите компьютер для применения всех изменений сетевого стека.
